# `docs/research/` — evidências internas (não públicas)

Para cada cidade `slug`, o gerador cria um arquivo `slug.md` com:

- fontes (URLs) usadas no passo de pesquisa (web search)
- notas de alta confiança (sem PII)

Uso: auditoria e rastreabilidade interna do porquê a página ficou “com cara de cidade”, sem precisar expor as fontes no HTML público.
